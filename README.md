°Jika NIM genap, maka kita hitung luas persegi dengan aturan berikut:


P = 2 digit NIM terakhir (yaitu 20)
L = 2 digit NIM pertama (yaitu 24)

Rumus luas persegi:
  luas=P×L=20×24=480

hasil luas persegi=480
